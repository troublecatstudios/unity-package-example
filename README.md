# Unity Package Example
An example GitHub Repository showing how to structure a Unity package project.

[![release][release-badge]][release]
[![license][license-badge]][license]

[Getting Started](./docs/README.md) | [Changelog](./package/CHANGELOG.md) | [License](./package/LICENSE.md)

**Supported Unity Versions**: v2021.3+

## Contributing

The project is broken out into two main directories `package` contains all the source files, dependencies, and documentation for the noir package. While `unity-project` is a Unity project that is setup to build out samples for inclusion in the final published version of the package.

For more info, see our [contribution guidelines](./CONTRIBUTING.md).
