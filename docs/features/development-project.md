# Development project

A Unity project is included in the Unity Package Example at the `./unity-project` folder. This project should be used to work on the package and build out any samples you'd like to include in your releases.

## Setting up the development project

1. Open Unity Hub and select "Add Project" -> "Add project from disk".
2. Select the `./unity-project` folder where you checked out the example project locally.
3. Open the project in the unity editor, you'll need to have at least Unity v2021.3.22f1 installed.

The unity project is already configured to pull in the package via a local filepath reference so once the project finishes importing you should be ready to start working on your package.
