    Unity Package Example is an example of how to create a new Unity Package project in GitHub. It includes many workflows and features that remove a lot of the headaches that make getting started with building Unity Packages difficult.

    Features

    • Automation – Releases, API documentation and more handled for you
    • Productivity – Build your package in Unity, alongside your samples, in the same repository
    • OpenUPM – Publish releases to OpenUPM automatically
    • Documentation – Documentation is published to GitHub Pages and powered by Material for mkdocs
